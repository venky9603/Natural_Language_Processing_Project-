# 🤖 NLP Rule-Based Chatbot (Streamlit)

This project is a simple **rule-based chatbot** built using **NLTK** and deployed with **Streamlit** for an interactive web interface.

---

## 🚀 Features

- 💬 Rule-based chatbot using regex patterns  
- ⚡ Fast response (optimized with Streamlit caching)  
- 🧠 Uses NLTK Chat utility  
- 🌐 Interactive UI with Streamlit  
- 🔄 Maintains chat history  

---

## 🛠️ Tech Stack

- Python  
- NLTK  
- Streamlit
